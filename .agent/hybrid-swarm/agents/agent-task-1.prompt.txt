# Task: Add A* grid pathfinding to terrain module

Implement an A* pathfinding function in src/world/terrain.rs that computes a walkable tile-level path between two positions, respecting is_passable(). The function signature should be roughly: pub fn find_path(&self, start: (u32,u32), goal: (u32,u32)) -> Option<Vec<(u32,u32)>>. Use 8-directional neighbors. Cost should incorporate the terrain speed_multiplier() so faster terrain is preferred. Return None if no path exists (goal unreachable). Also add a pub fn is_reachable(&self, start: (u32,u32), goal: (u32,u32)) -> bool convenience wrapper. Mountains (speed_multiplier=0.0) and Water (speed_multiplier=0.0) must be excluded from the search since is_passable() returns false for them. Keep the implementation simple — a BinaryHeap-based A* with Euclidean heuristic. No external crate dependencies.

Target files: src/world/terrain.rs
